Firefox 28.0

Mar 20th, 2014

セキュリティホール修正を含む firefox 28.0 がリリースされています．詳しくは以下をご覧ください．

  • Firefox 28.0 リリースノート

パッケージをリリースしました．sqlite, nss, nspr, libvpx パッケージも同時に Update する必要があります．

  • sqlite-3.8.4.1-i586-P1.txz (32bit)
  • nspr-4.10.4-i586-P1.txz (32bit)
  • nss-3.15.5-i586-P1.txz (32bit)
  • libvpx-1.3.0-i586-P1.txz (32bit)
  • firefox-28.0-i586-P1.txz (32bit)
  • sqlite-3.8.4.1-x86_64-P1.txz (64bit)
  • nspr-4.10.4-x86_64-P1.txz (64bit)
  • nss-3.15.5-x86_64-P1.txz (64bit)
  • libvpx-1.3.0-x86_64-P1.txz (64bit)
  • firefox-28.0-x86_64-P1.txz (64bit)
